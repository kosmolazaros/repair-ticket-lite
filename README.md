# Repair Ticket Lite

Mini web app (Vanilla JS) για καταχώρηση και διαχείριση tickets επισκευών (repair/service workflow).

## Status

✅ Completed up to **Day 2 – Task 2** (form structure + HTML validation)

## Current features

- Basic page layout (header + cards)
- "New Ticket" form with:
  - Labels + ids/names
  - HTML validation (required / minlength)
- Placeholder area for tickets list
- Simple JS smoke test + click counter

## Tech stack

- HTML
- CSS
- JavaScript (Vanilla)

## Run locally

1. Open the folder in VS Code
2. Run **Live Server** on `index.html`
